# 🧮 数学思维小课堂

面向小学 1-5 年级的数学思维训练交互式教学工具，将小学奥数经典题型转化为可视化、可操作的互动游戏。

## 特点

- **纯前端单文件**：打开 `index.html` 即可使用，无需安装或服务器
- **三阶段学习模型**：看一看（概念感知）→ 玩一玩（动手探索）→ 练一练（巩固练习）
- **可视化教具**：内置天平模型、条形图、数轴、韦恩图等交互组件
- **进度追踪**：星级评价系统，进度保存在浏览器本地
- **移动端适配**：响应式布局，支持手机和平板

## 涵盖知识点

| 年级 | 主题 | 核心能力 |
|------|------|----------|
| 🌱 一年级 | 找规律、数图形、移多补少、简单推理、排队问题 | 直观感知·简单规律 |
| 🌿 二年级 | 等量代换、周期问题、植树问题、归一问题、和差问题 | 等量关系·基础应用 |
| 🌳 三年级 | 和倍问题、差倍问题、年龄问题、盈亏问题、还原问题、平均数 | 倍数关系·逆向思维 |
| 🌲 四年级 | 鸡兔同笼、容斥原理、抽屉原理、逻辑推理、相遇问题、追及问题 | 经典模型·原理入门 |
| 🏆 五年级 | 流水行船、工程问题、牛吃草 | 综合应用·抽象建模 |

## 使用方式

直接双击 `index.html` 在浏览器中打开，或通过任意 HTTP 服务器托管。

## 技术栈

- 纯原生 HTML/CSS/JavaScript（无框架依赖）
- CSS 自定义属性实现主题配色
- localStorage 实现学习进度持久化
- SVG 实现天平模型等可视化组件
