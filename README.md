# 🧮 数学思维小课堂

面向小学 1-5 年级的数学思维训练交互式教学工具，将小学奥数经典题型转化为可视化、可操作的互动游戏。

## 特点

- **纯前端、零依赖**：原生 HTML/CSS/JS，无需安装任何框架或工具，打开即用
- **三阶段学习模型**：看一看（SVG 图解+奥数讲解）→ 玩一玩（滑块/拖拽探索）→ 练一练（200+ 题库自适应出题）
- **丰富的可视化**：天平模型、线段图、韦恩图、植树示意图、抽屉分配等 SVG 交互组件
- **教材来源标注**：25 个游戏均标注对应人教版教材单元，确保教学内容与教材一致
- **游戏化系统**：积分引擎（答对+10/连胜翻倍/探索+5）、11 个成就徽章、🔥连胜追踪
- **限时挑战模式**：练一练内可开启，根据难度限时（120s/75s/45s），全对额外加分
- **3 级分层提示**：温和提醒 → 策略引导 → 完整答案，渐进式脚手架
- **自适应难度**：连续答对 3 题自动升级（🟢基础→🟡进阶→🔴挑战），答错降级
- **Stage 1 教学动画**：步骤逐条揭示 + 点击交互双模式，关键公式弹入、SVG 图解入场动画
- **庆祝动画**：答对后大拇指 👍 + "你真棒！" 全屏弹入，强化正向激励
- **糖果色视觉**：柔和渐变背景、圆润卡片、明快配色，适合儿童审美
- **进度持久化**：积分、成就、星星全部保存在浏览器 localStorage
- **45 项自动化测试**：题库完整性、答案验证、框架函数全覆盖
- **移动端适配**：响应式布局，支持手机和平板

## 使用方式

由于浏览器安全策略（`localStorage` 需要 HTTP 环境），推荐通过任意 HTTP 服务器打开：

```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```

或直接双击 `index.html`（部分浏览器可能限制 localStorage）。

## 技术要点

- 纯原生 HTML / CSS / JavaScript（无框架、无构建工具）
- CSS 自定义属性（`--c1` ~ `--c6`）统一糖果色主题配色
- `checkAnswer()` 统一答案比较引擎：支持整数精确、浮点容差（0.15）、分数字符串、逗号对、对象映射
- `awardStage3()` / `penalizeStage3()` 共享评分函数：统一 Stage 3 正确/错误处理 + 进度追踪
- Stage 1 教学动画引擎：`animateStage1()` 支持自动播放 / 点击交互双模式
- localStorage 实现积分、成就、星星、游戏分数的持久化
- SVG 内联绘制天平、线段图、韦恩图等教学可视化组件
- `Intl.Segmenter` 级别注意：emoji 使用 `[...str]` 而非 `.split('')` 处理多码点字符
- 自适应难度引擎：根据题目在题库中的位置动态计算 easy/medium/hard 分级

## 题库设计

**核心原则**：所有题目完全遵守权威教材原文，不修改一个文字。确保教学内容与教材一致，避免自行改写引入错误或偏离教学大纲。

题库（`js/data.js`）与游戏逻辑（`js/games-grade1~5.js`）完全分离，新增题目只需在 `data.js` 中添加数据条目，无需修改任何逻辑代码。题目参考小学奥数经典教材（人教版、华罗庚学校、举一反三等），经逐题公式验证和 45 项自动化测试覆盖。
