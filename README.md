# 🧮 数学思维小课堂

面向小学 1-5 年级的数学思维训练交互式教学工具，将小学奥数经典题型转化为可视化、可操作的互动游戏。

## 特点

- **纯前端、零依赖**：原生 HTML/CSS/JS，无需安装任何框架或工具，打开即用
- **三阶段学习模型**：看一看（SVG 图解+奥数经典讲解）→ 玩一玩（滑块/拖拽探索）→ 练一练（200+ 题库随机出题）
- **丰富的可视化**：天平模型、线段图、韦恩图、植树示意图、抽屉分配等 SVG 交互组件
- **进度追踪**：每个游戏 3 星评价，进度保存在浏览器 localStorage
- **移动端适配**：响应式布局，支持手机和平板

## 项目结构

```
teaching/
├── index.html          # 入口页面（HTML 骨架）
├── css/
│   └── style.css       # 全部样式（CSS 变量、布局、动画）
├── js/
│   ├── data.js         # 题库数据（25 个游戏 × 8-10 题 = 200+ 题）
│   ├── framework.js    # 核心框架（导航、进度管理、共享工具函数）
│   └── games.js        # 游戏逻辑（25 个游戏的 stage1/2/3 + 判定）
└── README.md
```

## 使用方式

由于浏览器安全策略（`localStorage` 需要 HTTP 环境），推荐通过任意 HTTP 服务器打开：

```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```

或直接双击 `index.html`（部分浏览器可能限制 localStorage）。

## 技术栈

- 纯原生 HTML / CSS / JavaScript（无框架、无构建工具）
- CSS 自定义属性（`--c1` ~ `--c6`）统一主题配色
- localStorage 实现进度持久化
- SVG 内联绘制天平、线段图、韦恩图等教学可视化组件

## 题库设计

题库（`js/data.js`）与游戏逻辑（`js/games.js`）完全分离，新增题目只需在 `data.js` 中添加数据条目，无需修改任何逻辑代码。题目参考小学奥数经典教材（华罗庚学校、举一反三等），经逐题人工验证。
